# react-learning
Reactの学習用リポジトリ。基本から練習コードをまとめています。

今回は、ターミナルとGitHubを活用して初めて自分のローカル環境からコードをアップロードしました。AIを活用しながらも、Gitの設定やエラー対応にはかなり苦戦し、それだけに達成感も大きかったです。このリポジトリはまだ初期段階ですが、今後は実用的な機能の実装やUIの工夫を通じてレベルアップを目指していきたいです。

## 使用技術
- React
- JavaScript
- GitHub
- VS Code / ターミナル(macOS)
